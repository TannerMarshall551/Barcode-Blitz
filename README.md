# Barcode-Blitz
Repository for the senior design project for Barcode Blitz.

Keybinds:

MOVE - WASD

LOOK - Mouse

JUMP - Space

SPRINT - Left Shift

CROUCH - Left Control

TOGGLE SCANNER - E

PICKUP/DROP - Left Click or Q

SCAN LABEL - Right Click (maybe add another key bind?)

ENTER ROTATION MODE - R

ROTATE BOX (WHILE IN ROTATION MODE) - WASD
